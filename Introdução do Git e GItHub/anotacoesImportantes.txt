Anotações:

Untracked: são arquivos que o Git não tem ciência;
Tracked: arquivos que o Git tem ciência;
 -Unmodified:Quando o arquivo não foi modificado.
 -Modified:Arquivo que sofreu alteração.
 -Stage:Arquivos que estão sendo preparados para o commit.
